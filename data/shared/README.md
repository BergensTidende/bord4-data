# Delte saksdatasett

Legg publiserbare datasett fra enkeltsaker i en egen undermappe:

```text
data/shared/<saksslug>/
```

Hver mappe bør ha:

- en kort `README.md` med kilde, metode, kontaktpunkt og publiseringsdato
- data i CSV eller JSON
- scripts/notebooks bare hvis de trengs for å forstå eller reprodusere datasettet

Ikke legg rådata med sensitive opplysninger her.
